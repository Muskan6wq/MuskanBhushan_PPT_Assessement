TASK 1 – Static Website Hosting in EC2 (Ubuntu + Nginx)

1. Launch Ubuntu EC2 instance
2. Allow SSH (22) and HTTP (80) in Security Group
3. SSH into EC2 instance
4. Run sudo apt update
5. Run sudo apt install nginx -y
6. Run cd /var/www/html
7. Run sudo wget https://www.tooplate.com/download/2150_living_parallax
8. Run sudo mv 2150_living_parallax 2150_living_parallax.zip
9. Run sudo apt install unzip -y
10. Run sudo unzip 2150_living_parallax.zip
11. Run sudo mv 2150_living_parallax/* /var/www/html/
12. Run sudo rm -rf 2150_living_parallax 2150_living_parallax.zip

13. Run sudo systemctl restart nginx
14. Open browser and check
