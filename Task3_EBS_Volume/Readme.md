Objective:
Understand shared storage using Amazon EFS and mount it on EC2 instances.

Description:
Amazon EFS (Elastic File System) is a managed file storage service that can be shared across multiple EC2 instances. In this task, an EFS file system was created, mount targets were configured, and the file system was mounted on EC2 using NFS.

Steps Overview:
1.ﻗ·  ﻗ· EFS file system created.
2.ﻗ·  ﻗ· Mount targets configured in availability zones.
3.ﻗ·  ﻗ· Security group updated to allow NFS traffic.
4.ﻗ·  ﻗ· NFS client installed on EC2.
5.ﻗ·  ﻗ· EFS mounted on EC2.

Result:
EFS was successfully mounted and shared storage was accessible from EC2.
