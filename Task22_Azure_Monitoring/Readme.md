README – Task 22: Azure Monitoring & Diagnostics

Task Name:
Azure Monitoring and Diagnostics

Objective:
To monitor Azure resources and understand performance and activity logs using Azure Monitor.

Description:
Azure Monitor is used to collect and analyze performance data and logs from Azure resources such as virtual machines.

Steps Performed:
1. Logged in to Azure Portal.
2. Opened an existing Azure Virtual Machine.
3. Navigated to Monitoring section.
4. Viewed Metrics such as CPU Percentage.
5. Observed performance graphs over time.
6. Opened Activity Log to view actions like VM start and stop.
7. Explored Azure Monitor service for centralized monitoring.

Key Concepts Learned:
- Metrics show performance data like CPU and network usage.
- Activity Log records actions performed on Azure resources.
- Azure Monitor provides health and performance visibility.

Conclusion:
This task helped in understanding how Azure monitors resources and tracks performance and activity using metrics and logs.
