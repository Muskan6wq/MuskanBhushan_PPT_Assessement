README – Task 21: Azure Entra ID (Users, Roles, Identity)

Task Name:
Azure Entra ID (Identity and Access Management)

Objective:
To understand how identity and access management works in Azure using Azure Entra ID.

Description:
Azure Entra ID is used to manage identities and control access to Azure resources. It helps in managing users, roles, and permissions securely.

Steps Performed:
1. Opened Azure Portal.
2. Navigated to Azure Entra ID.
3. Viewed tenant and domain information.
4. Explored Users section in Entra ID.
5. Checked existing user account details.
6. Explored Roles and administrators section.
7. Understood role-based access control (RBAC) concepts.

Key Concepts Learned:
- Azure Entra ID manages identities in Azure.
- Users represent people who can log in.
- Roles define what actions a user can perform.
- RBAC is used to control access to Azure resources.

Conclusion:
This task helped in understanding Azure identity management, user roles, and access control using Azure Entra ID.
