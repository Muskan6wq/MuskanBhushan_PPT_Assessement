README – Task 23: Azure Resource Manager & Cost Management

Task Name:
Azure Resource Manager & Cost Management (Easy)

Objective:
Understand how Azure manages resources using Resource Groups and how to monitor and control cloud cost using Cost Management and Budgets.

Description:
In this task, a Resource Group was used to understand how Azure Resource Manager (ARM) works. ARM helps in managing, deploying, and organizing Azure resources. Cost Management was explored to view usage and cost details. A budget was also created to get alerts when spending reaches a defined limit.

Steps Performed:
1. Logged in to Azure Portal.
2. Opened Resource Groups from the Azure services.
3. Created a new Resource Group with a selected subscription and region.
4. Learned that all Azure resources are managed and deployed through Azure Resource Manager (ARM).
5. Opened Cost Management + Billing from the Azure portal.
6. Viewed Cost Analysis to understand how Azure tracks cost by service and time.
7. Created a budget with a fixed monthly amount.
8. Configured an alert to receive email notification when budget usage reaches a threshold.

Key Learnings:
- Azure Resource Manager controls all Azure resources.
- Resource Groups act as logical containers for Azure services.
- Cost Management helps monitor cloud spending.
- Budgets help avoid unexpected charges by sending alerts.

Conclusion:
This task helped in understanding how Azure organizes resources and manages cloud cost efficiently. Resource Groups and Cost Management are essential for controlling infrastructure and expenses in Azure.
